Pack IA pour business de cartes Pokémon

Instructions d'installation :
1. Assurez-vous d'avoir Python 3 installé.
2. Installez les dépendances nécessaires :
   pip install flask pandas

Utilisation :
- formulaire_description.py : Lancez pour générer des descriptions eBay via une interface graphique.
- detection_bonnes_affaires.py : Exécutez pour détecter les cartes avec ≥ 20 % de réduction.
- dashboard.py : Lancez pour accéder à un dashboard web local (http://127.0.0.1:5000) permettant d'ajouter, afficher et supprimer des cartes depuis la base de données SQLite.

Base de données :
- Un fichier cartes.db est automatiquement créé pour stocker les cartes.
